# Sistema de Caronas Universitárias
Sistema web para cadastro e gestão de caronas entre alunos da PUC Minas.

## Tecnologias
- HTML5, CSS3, JavaScript
- MySQL (modelagem de dados)

## Funcionalidades
- Cadastro de motoristas e caronas
- Busca por horários e rotas
- Validação de dados dos usuários

## Como testar
1. Clone o repositório
2. Abra `index.html` no navegador
3. Teste cadastro e listagem de caronas
